1. Java Introduction

Java is a popular and powerful programming language, created in 1995. It is owned by Oracle, and more than 3 billion devices run Java.

Key Concept

Its defining feature is "Write Once, Run Anywhere" (WORA). Compiled Java code can run on all platforms that support Java without the need for recompilation.

Your First Java Program

HelloWorld.java
public class HelloWorld {
public static void main(String[] args) {
// Prints "Hello, World" to the terminal window.
System.out.println("Hello, World!");
}
}

Don't worry if this code looks complex right now, we will break it down line-by-line in the next lesson! The key takeaway is the structure of a class and the main method.
